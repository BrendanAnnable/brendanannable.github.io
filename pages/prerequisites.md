# Prerequisites
- [Home Assistant](https://www.home-assistant.io/)
- [HACS Addon](https://www.hacs.xyz/)

Addons
- [EMHASS Addon](https://github.com/davidusb-geek/emhass-add-on)

Integrations
- [Solcast Integration](https://github.com/BJReplay/ha-solcast-solar/tree/main) (requires account and site setup)
- [Sigenergy ModBus Integration](https://github.com/TypQxQ/Sigenergy-Local-Modbus) (requires modbus enabled by installer)

Amber Pricing
- [Amber2MQTT Addon](https://github.com/cabberley/amber2mqtt-addon) (requires amber api key + siteid)
or
- https://www.home-assistant.io/integrations/amberelectric/

Optional for pretty dashboard graphs:
- [ApexCharts Card](https://github.com/RomRider/apexcharts-card)
