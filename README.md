# EMHASS + Amber Electric (Jekyll Guide)

A GitHub Pages site describing how to set up EMHASS with Amber Electric in Home Assistant.

## Quick start
1. Click **Use this template** on GitHub or push this folder to a new repo `emhass-amber` under your account.
2. Enable **Pages**: Settings → Pages → Build and deployment → Deploy from a branch → `gh-pages` (workflow below handles it).
3. The site will publish at `https://YOUR_GITHUB_USERNAME.github.io/emhass-amber`.

## Local preview
```bash
bundle install
bundle exec jekyll serve
```
