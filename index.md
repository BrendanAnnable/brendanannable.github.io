# EMHASS + Amber Electric

This site walks you through integrating **EMHASS** (Energy Management for Home Assistant) with **Amber Electric's** live wholesale prices in Australia, so you can automatically optimize when to charge/discharge your battery, run loads, or export to the grid.

> **Who is this for?**  
> Home Assistant users on Amber Electric who want a reproducible, transparent optimization pipeline using EMHASS.

## What you'll set up
- Amber price sensors in Home Assistant
- EMHASS connected to Home Assistant with an optimization config
- Schedules / automations to run EMHASS periodically
- Optional load & PV forecasts (e.g., Solcast) to improve decisions
